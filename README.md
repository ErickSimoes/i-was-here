## Atividades Práticas de Git e GitHub

> Introdução à Computação | CESAR School

Bem-vindos à nossa atividade prática com Git e GitHub! Este guia passo a passo foi criado para que vocês aprendam na prática como criar repositórios, colaborar com outros desenvolvedores e contribuir para projetos Open Source usando _Pull Requests_ (PRs).

Siga as instruções com atenção e não hesite em pedir ajuda aos colegas, monitores ou professores caso tenha dúvidas.

## 🛠️ Pré-requisitos para a Atividade

Antes de iniciar os passos descritos no guia, certifique-se de que você possui:

1. Uma conta ativa no [GitHub](https://github.com);
2. O Git devidamente instalado e configurado na sua máquina local;
3. Acesso ao seu e-mail institucional da **School** (necessário para a identificação na Fase 3).
